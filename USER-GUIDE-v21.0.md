# User Guide — v21.0 Commercial SaaS

## What is AI Governance Hub?

An **Enterprise AI Governance Assessment Platform**. Upload project data from Jira, Azure DevOps, Excel, or CSV. Receive an executive assessment with governance score, AI candidates, risk analysis, and framework mapping.

## Getting started

1. Visit [Start Assessment](https://aigovernancehub.ai/pricing.html)
2. Choose your platform (Jira, Azure DevOps, Excel, CSV)
3. Follow export instructions and download samples if needed
4. Upload your file — compatibility check runs automatically
5. Review free preview (score, candidates, risks, frameworks)
6. Review recommended plan and order summary
7. Confirm order and pay via Razorpay
8. Download report, receive email, access via [My Reports](https://aigovernancehub.ai/login.html)

## Order summary

Before payment you see: plan, base price, convenience fee (if any), taxes, and **total payable**. This total exactly matches Razorpay.

## Currency

Select INR, USD, EUR, GBP, AUD, or SGD. All prices come from the server — never hardcoded in the page.

## What you receive

- Executive PDF-ready assessment
- Governance score with explanation
- AI candidate inventory
- Risk matrix and recommendations
- Framework mapping (EU AI Act, ISO 42001, NIST AI RMF)
- Email delivery and recovery access
