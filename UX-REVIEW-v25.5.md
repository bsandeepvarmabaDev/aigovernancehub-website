# UX REVIEW — v25.5 Enterprise Launch

**Reviewed as:** Fortune 500 CIO, UX expert, enterprise customer  
**UX Score: 88/100**

## Executive summary

v25.5 addresses the primary enterprise hesitation points: unclear data handling, weak trust pages, internal-facing support copy, and pricing inconsistency. The journey now reads as premium enterprise SaaS.

## Screen-by-screen findings

| Screen | Before | v25.5 improvement |
|--------|--------|-------------------|
| Homepage | Generic lead; static ₹199 | Board-ready language; server-validated reassurance; dynamic pricing |
| Features | Orphan page, thin copy | Full chrome; six enterprise pillars; clear CTAs |
| Pricing | Technical upload note | Confidential upload + Trust Center links |
| Trust Center | Sparse, no nav | Procurement-grade security documentation |
| Refund Policy | 3 sentences, no chrome | Structured policy for CFO/procurement |
| Support | "Marketplace listing URL" note | Professional SLAs; Request ID guidance |
| Enterprise checkout | "Pay securely" | Secure Checkout + refund/support |
| Dashboard | Casual headline | Professional workspace intro |

## Copy principles applied

- Lead with outcomes (board-ready, executive formats) not mechanics
- Every error path: what / why / fix (retained from v25.3+)
- Removed internal implementation language from customer-facing pages
- Consistent "Secure Checkout" and "Enterprise Assessment" terminology

## Remaining UX gaps (post-launch)

1. Export walkthrough videos still placeholders
2. Index pricing card depends on `/api/pricing` — fallback if API unavailable
3. Cookie policy page still minimal chrome
