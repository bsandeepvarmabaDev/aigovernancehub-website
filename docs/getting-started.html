Getting Started Guide

This guide explains how customers begin using AI Governance Hub after installing the app from Atlassian Marketplace.

1. Install the App

Install AI Governance Hub from Atlassian Marketplace into the required Jira Cloud site. The Marketplace listing provides the installation entry point after approval.

2. Grant Required Permissions

Approve the requested app permissions so the Forge app can provide governance dashboard, review workflow, issue creation, evidence, and reporting functionality.

3. Open AI Governance Hub

Open Jira, go to Apps, and select AI Governance Hub. The app loads the governance dashboard experience inside Jira Cloud.

4. Configure Governance Settings

  • Review admin settings.
  • Confirm governance review categories.
  • Set expected operating model and review owners.
  • Confirm notification and reporting expectations.

5. Start a Governance Review

Register or review an AI use case, assess risk, route review actions, create Jira risk tickets where required, and track evidence and approval status.
