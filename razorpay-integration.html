Razorpay Payment Integration Plan

Use this page as the implementation checklist for converting paid assessment visitors into verified customers.

Open Assessment View Sample Report

Target Payment Flow

1. Create Order

Website sends assessment id, customer email, amount and currency to backend.

2. Razorpay Checkout

Customer pays using domestic or international payment method supported on your Razorpay account.

3. Verify Signature

Backend validates razorpay_order_id, razorpay_payment_id and razorpay_signature before unlocking report.

4. Deliver Report

Generate PDF, email report, and store lead/payment metadata.

Required Backend Endpoints

POST /api/create-razorpay-order
POST /api/verify-razorpay-payment
POST /api/send-assessment-report

Do not expose Razorpay key secret in this static website. Signature verification must run on a backend or serverless function.
